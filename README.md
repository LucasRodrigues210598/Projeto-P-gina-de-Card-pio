   # 🍕 La Fornace · Cardápio Digital Interativo

Cardápio digital para pizzaria artesanal com carrinho de compras e integração com WhatsApp. Projeto desenvolvido com HTML, CSS e JavaScript puro.

## ✨ Funcionalidades

- 📋 Cardápio dividido em categorias: Tradicionais, Especiais, Doces, Bebidas e Combos
- 🍕 Seleção de tamanho com preço dinâmico (Pequena, Média, Grande, Família)
- 🛒 Carrinho de compras com adição e remoção de itens
- 💬 Finalização de pedido direto pelo WhatsApp
- 🏷️ Badges de destaque: Top, Novo, Vegetariano
- 📱 Layout responsivo

## 🛠️ Tecnologias

- HTML5 semântico
- CSS3 (variáveis CSS, grid, flexbox, transições)
- JavaScript puro (DOM, arrays, template literals)

## 📁 Estrutura

```
cardapio-pizzaria/
├── index.html   → estrutura da página
├── style.css    → estilos e layout
└── script.js    → lógica do cardápio e carrinho
```

## 🚀 Como usar

1. Clone o repositório:
```bash
git clone https://github.com/LucasRodrigues210598/cardapio-pizzaria.git
```

2. Abra o arquivo `index.html` no navegador — sem necessidade de servidor!

3. Para personalizar, edite o número do WhatsApp em `script.js`:
```javascript
window.open("https://wa.me/55SEU_NUMERO?text=" + msg, "_blank");
```
